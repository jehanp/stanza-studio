# stanza-studio

A creative project that transforms Bible scripture into subtle, deeply meaningful song lyrics.

## Scripture-to-Lyrics Plan

See [SCRIPTURE-LYRICS-PLAN.md](SCRIPTURE-LYRICS-PLAN.md) for the full bi-weekly conversion schedule (April–October 2026), including:

- 14 songs mapped to prominent Bible books and chapters
- Lyric direction and subtle imagery guidance for each entry
- Music genre suggestions for every song
- A step-by-step methodology for converting scripture to lyrics

## Lyrics

Completed lyrics are stored in the [`lyrics/`](lyrics/) directory.
